# RNF-004

Categoría: Accesibilidad
Criterio de Medición: Auditoría WCAG 2.1 AA: contraste ≥4.5:1; 100% de funciones críticas navegables por teclado; etiquetas/alternativas textuales presentes; 0 bloqueos en pruebas con lector de pantalla (NVDA/VoiceOver).
Descripción: El sistema debe cumplir WCAG 2.1 AA para que los casos (y sus herramientas/recursos) sean accesibles por teclado y lectores de pantalla.
Estado: Propuesto
Fuente de Elicitación: Revisión normativa
Nombre del Requisito: Accesibilidad WCAG 2.1 AA
Prioridad: Alta
