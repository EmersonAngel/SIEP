# RNF-010

Categoría: Ética (contenidos sensibles)
Criterio de Medición: Incluye: advertencia previa al inicio, guía de lenguaje, botón de salida segura en todas las fases y recursos de apoyo; checklist ético con aprobación experta ≥95%; 0 ocurrencias de lenguaje estigmatizante en revisión de contenido publicada.
Descripción: El sistema debe incorporar advertencias, lineamientos de lenguaje y controles de pausa/salida segura para cualquier caso con contenido sensible, configurable por caso.
Estado: Propuesto
Fuente de Elicitación: Validación experta
Nombre del Requisito: Gestión ética de contenidos sensibles
Prioridad: Alta
