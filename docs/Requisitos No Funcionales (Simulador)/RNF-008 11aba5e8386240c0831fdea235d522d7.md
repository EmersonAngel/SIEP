# RNF-008

Categoría: Mantenibilidad
Criterio de Medición: ≥80% de ajustes de contenido se realizan sin despliegue de código; versionado con historial y rollback; pruebas de regresión pasan ≥95% tras actualización; documentación técnica actualizada por versión.
Descripción: El sistema debe gestionar el contenido de casos de forma modular y versionada para permitir ampliación del catálogo sin despliegues frecuentes.
Estado: Propuesto
Fuente de Elicitación: Análisis documental
Nombre del Requisito: Contenido modular y versionado
Prioridad: Media
