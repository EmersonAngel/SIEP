# RNF-001

Categoría: Usabilidad
Criterio de Medición: En prueba con usuarios (n≥10), ≥90% completa el inicio y el primer nodo sin ayuda; tiempo de localización de funciones clave (herramientas, ayuda, salir) ≤30 s; tasa de error crítico de navegación ≤5%.
Descripción: La interfaz del simulador debe ofrecer navegación guiada y consistente entre casos (mismo patrón de UI para escenas, decisiones, herramientas y retroalimentación) para reducir carga cognitiva.
Estado: Propuesto
Fuente de Elicitación: Entrevista docente
Nombre del Requisito: Navegación guiada y consistencia multi-caso
Prioridad: Alta
