# RNF-006

Categoría: Disponibilidad
Criterio de Medición: Disponibilidad mensual ≥99.5% en periodos académicos; RTO ≤4 h; notificación de incidentes ≤30 min; mantenimiento planificado comunicado con ≥48 h.
Descripción: El sistema debe mantener disponibilidad definida durante ventanas académicas y permitir continuidad de intentos.
Estado: Propuesto
Fuente de Elicitación: Entrevista docente
Nombre del Requisito: Disponibilidad en periodos académicos
Prioridad: Alta
