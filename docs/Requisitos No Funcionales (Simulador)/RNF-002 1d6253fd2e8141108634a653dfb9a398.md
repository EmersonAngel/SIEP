# RNF-002

Categoría: Seguridad
Criterio de Medición: Pruebas de autorización: 100% de acciones sensibles requieren rol; intentos de acceso indebido resultan en denegación y registro de auditoría; 0 hallazgos críticos en revisión de permisos.
Descripción: El sistema debe aplicar control de acceso por rol y mínimo privilegio para funciones de autoría, analíticas, exportaciones y administración del catálogo de casos.
Estado: Propuesto
Fuente de Elicitación: Revisión normativa
Nombre del Requisito: Control de acceso por rol y mínimo privilegio
Prioridad: Alta
