# RNF-003

Categoría: Privacidad
Criterio de Medición: Inventario de datos: 100% de campos justificados; bitácoras no se incluyen en reportes agregados por defecto; exportación de texto libre sensible requiere habilitación explícita; 0 exposiciones no autorizadas en pruebas.
Descripción: El sistema debe minimizar recolección y exposición de datos, anonimizar vistas agregadas y controlar exportaciones, especialmente para contenidos y textos libres potencialmente sensibles.
Estado: Propuesto
Fuente de Elicitación: Validación experta
Nombre del Requisito: Minimización y protección de datos
Prioridad: Alta
