# RNF-005

Categoría: Rendimiento
Criterio de Medición: En carga nominal, p95 de carga de nodo ≤2 s y apertura de herramienta ≤2.5 s; en carga alta (2× concurrencia), p95 ≤4 s; tasa de error de carga ≤1%.
Descripción: El sistema debe cargar escenas y abrir herramientas con latencias acotadas para no interrumpir la simulación.
Estado: Propuesto
Fuente de Elicitación: Caso de estudio
Nombre del Requisito: Tiempo de respuesta de escenas y herramientas
Prioridad: Media
