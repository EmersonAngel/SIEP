# RNF-007

Categoría: Trazabilidad
Criterio de Medición: Eventos mínimos registrados: inicio/cierre de intento, cada decisión confirmada, checklist completado, exportación y cambios; logs con sello de tiempo, usuario/rol y contexto; retención ≥12 meses; 100% consultables por administrador autorizado.
Descripción: El sistema debe registrar eventos académicos y de seguridad (decisiones, herramientas, cambios de configuración, exportaciones) para auditoría y mejora continua.
Estado: Propuesto
Fuente de Elicitación: Revisión normativa
Nombre del Requisito: Auditoría de eventos y accesos
Prioridad: Alta
