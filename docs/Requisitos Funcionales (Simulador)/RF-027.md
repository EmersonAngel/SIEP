# RF-027

Criterio de Aceptación: Dado un usuario autenticado con permisos válidos, cuando consulte el asistente de IA dentro de un caso o resultado del simulador, entonces el sistema debe generar una respuesta contextual relacionada con el caso, registrar la interacción y bloquear o advertir consultas fuera del alcance. La IA no debe mostrar la opción correcta antes de que exista una decisión registrada.
Descripción: El sistema debe permitir el uso de un asistente de inteligencia artificial integrado al simulador para orientar al estudiante y al docente mediante explicaciones conceptuales, preguntas de reflexión, aclaración de instrucciones, consulta de recursos teóricos y apoyo en la comprensión del desempeño, sin reemplazar la toma de decisiones del estudiante ni la evaluación docente.
Estado: Propuesto
Módulo: Retroalimentación
Nombre del Requisito: Asistencia inteligente contextual del simulador
Prioridad: Alta
Regla de Negocio: El asistente de IA solo podrá responder sobre contenidos, casos, herramientas, recursos y resultados asociados al simulador. No debe revelar respuestas correctas antes de que el estudiante tome una decisión, no debe emitir diagnósticos clínicos reales, no debe sustituir al docente y debe advertir cuando la consulta no esté relacionada con el objetivo formativo del sistema.
Tipo: Funcional
Usuarios: Estudiante, Docente
