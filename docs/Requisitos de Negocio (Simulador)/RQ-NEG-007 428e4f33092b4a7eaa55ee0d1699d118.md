Descripción: La solución debe permitir crear, versionar y reutilizar casos, herramientas y recursos para múltiples cohortes y asignaturas.
Estado: Propuesto
Fuente de Elicitación: Validación experta
Nombre del Requisito de Negocio: Escalar el catálogo con reutilización y versionamiento
Objetivo Institucional Relacionado: Sostenibilidad y escalabilidad de recursos educativos digitales.
Prioridad: Media
