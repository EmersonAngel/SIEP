# Revisión con equipo de ingeniería de software

Aporte al Levantamiento de Requisitos: Alinea requisitos con restricciones técnicas, fortalece requisitos no funcionales (seguridad, privacidad, rendimiento, escalabilidad) y define criterios verificables de implementación.
Información Esperada: Decisiones de arquitectura, modelos de datos, criterios de logging/auditoría, estrategias de pruebas, riesgos técnicos y plan de iteración.
Objetivo: Evaluar factibilidad técnica, arquitectura, mantenibilidad, trazabilidad y escalabilidad del simulador, asegurando que el diseño soporte analíticas, versionamiento de casos y control de datos sensibles.
Participantes: Equipo de ingeniería de software, líder de proyecto y representantes académicos para aclaraciones.
