# Prototipado de escenarios

Aporte al Levantamiento de Requisitos: Reduce ambigüedad, permite iterar requisitos funcionales y no funcionales, y mejora consistencia/claridad del flujo del simulador.
Información Esperada: Mapas de flujo, pantallas clave, textos de apoyo, puntos de fricción, tiempos de respuesta percibidos y necesidades de accesibilidad.
Objetivo: Explorar y evaluar tempranamente la interacción del estudiante con nodos del caso, uso de herramientas (PAP/SPIKES/riesgo) y activación de rutas, antes de implementar versiones completas.
Participantes: Equipo de diseño/ingeniería, docentes y estudiantes piloto (evaluación).
